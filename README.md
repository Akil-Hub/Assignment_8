 ## Qurbanir Hat

A modern online marketplace for buying and selling Qurbani cattle in Bangladesh. Browse verified animals, view detailed listings, and connect with trusted sellers — all in one place.

**Live URL:** https://qurbanir-hat20.vercel.app/



##  Key Features

 Browse all available Qurbani cattle with filters
 Detailed animal pages with images, weight, age, breed, and location
 User authentication — register, login, and manage your profile
 Update profile name and avatar
 Protected routes for profile and animal details
 Dark and light theme support
 Fully responsive design for mobile and desktop
 Animated hero banner
 Featured animals slider
